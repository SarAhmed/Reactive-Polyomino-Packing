Reactive Polyomino Packing (RPP): hover to display the controllers

Controllers:

  • Drag a tile ⇒ click and drag.

  • Enlarge a tile ⇒ hover on the tile using the mouse then press the up arrow on the keyboard.

  • Shrink a tile ⇒ hover on the tile using the mouse then press the down arrow on the keyboard.

  • Rotate clockwise a tile ⇒ hover on the tile using the mouse then press the right arrow on the keyboard.

  • Rotate anticlockwise a tile ⇒ hover on the tile using the mouse then press the left arrow on the keyboard.
